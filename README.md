Hello
=======

RPi + NodeJS + DigiSpark

See package.json for list of npm packages used.

To install:

1.  git clone git://github.com/niquels/webgpio
2.  npm install webgpio

To run:

1. node webgpio/app.js
2. in a browser goto the server:3000
